## Intro
Extract still JPEG and MP4 from Google Camera Motion Photos (e.g. from Google Pixel 2)

For Samsung devices: https://github.com/chlete/samsung-motion-photo-splitter

### Batch conversion

 ```bash                                                                        
 for p in *.jpg; do                                                            
    python splitter.py ${p}                                                     
 done                                                                          
 ```
